# 🚀 UPSC InsightX – Intelligent Mock & Analytics Engine

UPSC InsightX is a Python-powered intelligent mock test and analytics engine designed specifically for UPSC aspirants.

It enables users to practice subject-wise quizzes, analyze performance trends, detect weak areas, and generate professional PDF scorecards.

---

## 🎯 Key Features

- 📘 Subject-wise quizzes (History, Polity, Economy, Science & Technology)
- 🎯 Multiple difficulty levels (Easy, Medium, Hard)
- ✅ Real-time score calculation
- 🧠 Intelligent weakness detection system
- 📊 Performance trend analytics
- 📄 Automated PDF scorecard generation
- 👤 Secure user login system
- 🏆 Gamified experience with XP & Levels

---

## 🛠 Tech Stack

- Python
- Matplotlib (Analytics Visualization)
- ReportLab (PDF Generation)
- JSON (Data Storage)
